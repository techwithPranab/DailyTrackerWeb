# 🔄 Recurring Activities - Quick Reference Card

## Create Recurring Activity

### Daily Activity
```
Example: Morning Exercise
✓ Check "Make this a recurring activity"
✓ Select "Daily"
✓ Set start time
✓ Optional: Set end date
```

### Weekly Activity
```
Example: Piano Practice (Mon/Wed/Fri)
✓ Check "Make this a recurring activity"
✓ Select "Weekly"
✓ Click days: Mon, Wed, Fri
✓ Optional: Set end date
```

### Monthly Activity
```
Example: Monthly Review (1st of each month)
✓ Check "Make this a recurring activity"
✓ Select "Monthly"
✓ Set start date to 1st
✓ Optional: Set end date
```

## Weekly Tracker Usage

### Navigation
- **← Previous** = Go back one week
- **This Week** = Return to current week
- **Next →** = Go forward one week

### Mark Complete
- Click **○** (gray circle) → Changes to **✓** (green checkmark)
- Activity is marked complete for that specific day

### Mark Incomplete
- Click **✓** (green checkmark) → Changes to **○** (gray circle)
- Activity is marked incomplete for that day

## Visual Indicators

### Priority
- 🔴 **High** = Red badge
- 🟡 **Medium** = Yellow badge
- 🟢 **Low** = Green badge

### Category
- 🧹 **Chores**
- 📚 **School**
- 💪 **Fitness**
- 🎨 **Hobby**
- 📋 **Other**

### Status
- **○** = Not completed
- **✓** = Completed
- **-** = Not scheduled for this day

### Special
- **🔄 daily/weekly/monthly** = Recurring activity
- **Blue background** = Today's column

## Statistics Box

Shows at bottom of weekly tracker:
- **Total Activities** = Number of activity occurrences in week
- **Completed** = How many you've completed
- **Completion Rate** = Percentage complete

## Common Patterns

### School Schedule
```
Math Homework - Weekly (Mon, Wed, Fri)
Reading - Daily
Science Lab - Weekly (Thu)
```

### Fitness Routine
```
Morning Run - Daily
Gym Session - Weekly (Mon, Wed, Fri)
Yoga - Weekly (Tue, Thu, Sat)
```

### Household Chores
```
Make Bed - Daily
Clean Room - Weekly (Sat, Sun)
Laundry - Weekly (Wed)
```

## Tips

✅ **Use Daily** for habits you want to do every single day
✅ **Use Weekly** for activities on specific days
✅ **Use Monthly** for once-a-month tasks
✅ **Set end dates** for temporary recurring activities
✅ **Check completion rate** to track your progress
✅ **Navigate weeks** to plan ahead or review past

---

**Quick Access**: Dashboard → Scroll to "Weekly Activity Tracker"
